jui_datagrid documentation (v1.00)

jui_datagrid is an Ajax-enabled jQuery plugin, useful to manipulate data in tabular format.

As a javascript control, it can be integrated with any server-side technology (e.g. PHP, Perl, ASP, JSP and more).

Compatible with modern web browsers. Tested with: Google Chrome, Mozilla Firefox, Microsoft Internet Explorer, (default) Android browser.

jui_datagrid API is simple and powerful.

Project page: https://github.com/pontikis/jui_datagrid

  • jquery
  • jquery-ui
  • jquery-ui themes
  • jui_pagination plugin (documentation here)
  • jui_dropdown plugin (documentation here)
  • For touch event support jquery.ui.touch-punch.min.js could be used (see folder /lib//jui_pagination/lib)

Tested with jquery >= 1.8.2 and >= jquery-ui 1.8.23 (recommended jquery-ui >= 1.9)

Features

  • Diplay tabular data
  • CSS based themes compatible with jQuery-UI Theming
  • Get data in JSON format using AJAX (any server-side technology)
  • Sort columns data
  • Change columns order
  • Show/hide columns
  • Style columns
  • Single or multi-select rows
  • Powerful pagination
  • Preferences dialog available to user
  • Localization

Usage

    <link rel="stylesheet" href="/path/to/jquery.jui_datagrid.css">

    <script type="text/javascript" src="/path/to/jquery.js" type="text/javascript"></script>
    <script type="text/javascript" src="/path/to/jquery-ui.js" type="text/javascript"></script>

    $("#element_id").jui_datagrid({

Methods

COMMON

getDefaults

    $("#element_id").jui_datagrid('getDefaults');

getOption

    $("#element_id").jui_datagrid('getOption', 'option_name');

getAllOptions

    $("#element_id").jui_datagrid('getAllOptions');

setOption

    $("#element_id").jui_datagrid('setOption', 'option_name', option_value, reinit);

Alternative way to set one or more options 

    $("#element_id").jui_datagrid({option1_name: option1_value, etc});

refresh

    $("#element_id").jui_datagrid('refresh');

destroy

    $("#element_id").jui_datagrid('destroy');

Events

onCellClick

    $("#element_id").jui_datagrid({
        onCellClick: function(event, data) {
            // your code here e.g.
            console.log('Click on cell: col ' + data.col + ' row ' + data.row + '.');
        }
    });

data is an object with properties: col and row (the index of the column and row).

onRowClick

    $("#element_id").jui_datagrid({
        onRowClick: function(event, data) {
            // your code here e.g.
            console.log('Row with ID ' + data.row_id + ' ' + data.row_status + '.');
        }
    });

data is an object with properties: row_id and row_status ('selected', 'deselected').

onDelete

    $("#element_id").jui_datagrid({
        onDelete: function() {
            // your code here e.g.
            var a_sel = $(this).jui_datagrid("getSelectedIDs"),
                sel = a_sel.length;
            if(sel == 0) {
                console.log('Nothing selected...');
            } else {
                console.log(sel + ' Row(s) with ID: ' + a_sel + ' will be deleted.');
            }
        }
    });

onDisplay

    $("#element_id").jui_datagrid({
        onDisplay: function() {
        // your code here
        }
    });
