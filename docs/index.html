jui_datagrid documentation (v1.00)

jui_datagrid is an Ajax-enabled jQuery plugin, useful to manipulate data in tabular format.

As a javascript control, it can be integrated with any server-side technology (e.g. PHP, Perl, ASP, JSP and more).

Compatible with modern web browsers. Tested with: Google Chrome, Mozilla Firefox, Microsoft Internet Explorer, (default) Android browser.

jui_datagrid API is simple and powerful.

Project page: https://github.com/pontikis/jui_datagrid

Requires

  • jquery
  • jquery-ui
  • jquery-ui themes
  • jui_pagination plugin (documentation here)
  • For touch event support jquery.ui.touch-punch.min.js could be used (see folder /lib//jui_pagination/lib)

Tested with jquery 1.8.2 and jquery-ui 1.9.0

Features

  • Diplay tabular data
  • Get data in JSON format using AJAX (any server-side technology)
  • Powerful pagination
  • CSS based themes compatible with jQuery-UI Theming

Usage

HTML (head section)

    <link rel="stylesheet" href="/path/to/jqueryui_theme/jquery-ui.css">
    <link rel="stylesheet" href="/path/to/jquery.jui_pagination.css">
    <link rel="stylesheet" href="/path/to/jquery.jui_datagrid.css">
    <!--  if custom classes used -->
    <link rel="stylesheet" href="/path/to/custom.css">

    <script type="text/javascript" src="/path/to/jquery.js" type="text/javascript"></script>
    <script type="text/javascript" src="/path/to/jquery-ui.js" type="text/javascript"></script>

    <!--  if touch event support for touch devices is needed (see folder /lib) -->
    <script src="/path/to/jquery.ui.touch-punch.min.js" type="text/javascript"></script>

    <script type="text/javascript" src="/path/to/jquery.jui_pagination.js" type="text/javascript"></script>
    <script type="text/javascript" src="/path/to/jui_pagination/localization/en.js" type="text/javascript"></script>

    <script type="text/javascript" src="/path/to/jquery.jui_datagrid.js" type="text/javascript"></script>
    <script type="text/javascript" src="/path/to/jui_datagrid/localization/en.js" type="text/javascript"></script>

JS code

    $("#element_id").jui_datagrid({
        ajaxFetchDataURL: 'ajax/ajax_fetch_data1.php',
        containerClass: 'grid1_container ui-state-default ui-corner-all',
        title: 'Customers',
        paginationOptions: {
            sliderAnimation: false
        }
    });

        // DEFAULTS
        pageNum: 1,
        rowsPerPage: 10,
        maxRowsPerPage: 100,

        // toolbar options
        useToolbar: true,
        showPrefButton: true,
        showSelectButtons: true,
        showSelectLabel: false,
        showRefreshButton: true,
        showDeleteButton: true,
        showPrintButton: true,
        showExportButton: true,
        showFiltersButton: true,
        showPrefButtonText: false,
        showSelectAllButtonText: false,
        showSelectNoneButtonText: false,
        showSelectInverseButtonText: false,
        showRefreshButtonText: false,
        showDeleteButtonText: true,
        showPrintButtonText: false,
        showExportButtonText: false,
        showFiltersButtonText: false,

        // pagination options
        usePagination: true,

Methods

COMMON

getDefaults

    $("#element_id").jui_datagrid('getDefaults');

getOption

    $("#element_id").jui_datagrid('getOption', 'option_name');

getAllOptions

    $("#element_id").jui_datagrid('getAllOptions');

setOption

    $("#element_id").jui_datagrid('setOption', 'option_name', option_value, reinit);

Alternative way to set one or more options 

    $("#element_id").jui_datagrid({option1_name: option1_value, etc});

refresh

    $("#element_id").jui_datagrid('refresh');

destroy

    $("#element_id").jui_datagrid('destroy');
STYLING

setGridStyle

    $("#element_id").jui_datagrid('setGridStyle', tableClass, trHoverClass, thClass, tdClass, trLastClass);
PAGINATION

setPaginationOptions

    $("#element_id").jui_datagrid('setPaginationOptions', pag_options);

getPaginationOption

    $("#element_id").jui_datagrid('getPaginationOption', 'option_name');

getAllPaginationOptions

    $("#element_id").jui_datagrid('getAllPaginationOptions');
